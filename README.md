# Sitio web sencillo de Fórmula 1

Este proyecto es un sitio estático con información básica sobre la Fórmula 1. Tiene tres páginas principales:

- **Inicio**: muestra equipos, calendario 2026, pilotos y circuitos icónicos.
- **Historia**: ofrece un resumen histórico, listados de circuitos históricos, campeones del mundo y algunas imágenes.
- **Contacto**: formulario para que los visitantes puedan enviar comentarios.

## Características

- HTML, CSS y JavaScript puros.
- Diseño con fondo claro, encabezados rojos y navegación oscura.
- Validación de formulario básica y eventos interactivos mediante JavaScript.

## Cómo usar

1. Abre `index.html` en tu navegador (doble clic o arrastra al navegador).
2. Navega por las páginas mediante el menú superior.

> El proyecto está pensado para publicarse en GitHub Pages. Coloca los archivos en el repositorio y activa Pages en la configuración.
